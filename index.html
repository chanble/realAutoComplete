Welcome to chanble's project Realautocomplete.

This is a auto complete text input jquery plugin.

You can install it like this:

1,
download it to you documentroot directory.
2, 
http://localhost/realAutoComplete/test/autoComplete.html

docurmnt:


	var da = ['one' ,'tow' ,'three' ,'four'
			,'five', 'six', 'seven', 'eight'
			, 'nine', 'ten', 'one1', 'two1'
			,'three1', 'four1'];
	$("#ac").RealAutoComplete({matchModel:'every', data:da, maxNum: 3});

Or you can get data from web server:


$("#ac").RealAutoComplete({url:'your url (e.g localhost/server.php)'
							,matchModel:'every'//pre, every
							, data:da, maxNum: 3 //number of show
						});
